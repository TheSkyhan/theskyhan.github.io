Overview

BrandLedger is a creator business operations app for tracking deals, contracts, invoices, reminders, and subscription access. This version of BrandLedger stores workspace content locally on the user’s device.

Information we collect

  • Workspace profile details entered by the user, such as creator name, niche, location, and revenue goal
  • Deal, contract, and invoice records created by the user
  • Local subscription entitlement state returned by Apple StoreKit
  • Local notification preferences used for reminder scheduling

How information is used

  • To render dashboards, analytics, and creator workflow screens
  • To manage subscription access and plan-based limits
  • To schedule reminders the user explicitly enables
  • To support backup export and import features on supported plans

Data sharing

BrandLedger does not sell user data. The current version does not include third-party advertising SDKs or third-party analytics SDKs. Subscription processing is handled by Apple through StoreKit.

Data storage

User-entered workspace content is stored locally on the device in the current version of BrandLedger. If a user exports a backup file, that file is stored wherever the user chooses to save it.

Tracking

BrandLedger does not track users across apps or websites owned by other companies.

User controls

  • Users can edit or delete workspace records inside the app
  • Users can export or import workspace backups on supported plans
  • Users can disable notifications in iOS or iPadOS settings
  • Users can manage subscriptions through their Apple ID subscription settings

Children’s privacy

BrandLedger is not directed to children under 13.

Contact

For privacy questions, contact support@brandledger.app.
