Frequently asked questions

How do I restore my subscription?

Open Growth Lab and tap Restore Purchases. Make sure you are signed in with the same Apple ID used for the original purchase.

Where is my data stored?

BrandLedger currently stores workspace data locally on your device using SwiftData. You can export a backup from Settings and re-import it on supported plans.

Why are reminders not appearing?

Reminder notifications are available when the plan supports reminders and notification permission has been granted in Settings. iOS notification settings can also block delivery.

Can I use BrandLedger without a subscription?

Yes. The free workspace supports a limited number of deals, invoices, and contracts so you can try the workflow before upgrading.

How do I move my workspace to another device?

Export a workspace backup from Settings, then import that file on the new device if your plan supports imports.

App review reference

This app does not require account creation or sign-in for core use. Notification permission is requested only when the user explicitly enables reminders from Settings.
